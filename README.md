# Weathermar

A mobile weather app built with React Native and Expo.

This project uses the Open-Meteo API to fetch and display forecast data. I built it to practice API integration, state management, reusable UI components, and mobile app structure.

## Features

- Fetch weather forecast data from a public API
- Display current and forecast weather information
- Search for locations
- Use reusable UI components
- Show weather-related visual elements

## Tech Stack

- React Native
- Expo
- TypeScript
- Open-Meteo API

## What I Practiced

- Fetching data from an external API
- Managing state in a React Native app
- Building reusable components
- Handling API response data
- Structuring a mobile app project

## Status

In progress. This project is being used as a learning project for React Native, API-driven applications, and mobile UI development.
